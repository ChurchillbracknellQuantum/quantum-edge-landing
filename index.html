Common Questions

What broker should I use with TITAN?
+
TITAN works with any MT5 broker offering the 5 pairs. We backtested on Exness MT5. Low-spread brokers recommended — under 10 pips per pair. Exness, Pepperstone, and IC Markets are all suitable.
What is the minimum account size?
+
Minimum recommended is $500 per EA, or $2,000–$2,500 if running all 5 pairs simultaneously. We backtested on $2,000 starting capital.
Can I run all 5 pairs at the same time?
+
Yes — and this is the recommended setup. Each pair uses an independent magic number and risk percentage. Set RiskPercent to 0.3–0.5% per pair when running all 5 together.
Does TITAN work on a Prop Firm account?
+
Yes — TITAN AutoTrader Pro is specifically engineered for prop firms. Complies with FTMO (5% daily / 10% overall), FundedNext, and Infinity Forex Funds rules including 4% daily DD hard stop and news blackout.
Is my license key transferable?
+
No. Each license key binds permanently to one MT5 account number on first activation and cannot be transferred. Contact support if you need to change accounts.
Do I need coding knowledge?
+
None. Installation takes under 5 minutes: copy the file to your Experts folder, press F7 to compile, attach to chart, enter your license key. Full step-by-step guide included.
Do I need MT5 running 24/7?
+
Yes — TITAN needs MT5 running continuously. We recommend a VPS starting at $5–10/month. Hostinger VPS, Contabo, or ForexVPS.net all work well.
What pairs does TITAN NOT trade?
+
TITAN is optimised for USDCHF, GBPUSD, AUDUSD, AUDCHF, and CADCHF only. All JPY pairs, EURUSD, and USDCAD were permanently eliminated after testing. Do not add other pairs.
How many trades per month?
+
Running all 5 pairs, TITAN averages approximately 11–12 trades per month. Quality over quantity — fewer, higher-probability trades at 70–79% win rate outperforms high-frequency systems.

All sales are final on digital products. We provide verified backtest data and transparent performance results before purchase for fully informed decisions. Technical issues resolved promptly by support.

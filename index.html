What broker should I use?+

Any MT5 broker with Raw/Pro spread accounts. Exness, IC Markets, Pepperstone all work well. Avoid Standard accounts — spread eats profits. Max spread: 20 points on forex pairs.

What account size do I need?+

Minimum $500, recommended $1,000–$2,000. Backtest run on $2,000. Running all 5 pairs: set RiskPercent to 0.3% per pair (max 1.5% total exposure).

Does it work on FTMO and FundedNext?+

Yes — TITAN Pro is built for prop firm challenges. 4% daily DD hard stop (below FTMO's 5% rule). News blackout handles FOMC/NFP automatically. Buyer guide includes exact input settings per firm.

Do I need to watch it 24/7?+

No. TITAN runs fully automated on a VPS. It monitors markets 24/5, manages all open trades, applies trailing stops, and closes everything Friday before market close.

Is this a martingale or grid system?+

No. Fixed lot sizing only. One position maximum per pair at any time. No doubling on losers. Hard daily DD cap and overall DD cap built in as code-level hard stops.

What VPS do I need?+

Any Windows VPS with MT5 installed. Hostinger VPS (~$8/mo), Contabo (~$5/mo), or ForexVPS.net work well. You host the VPS — TITAN runs inside your MT5 terminal.
